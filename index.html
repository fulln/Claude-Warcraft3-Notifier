Details

Tuned for Real Work

A stripped-down, reliable, and dependency-free asset designed for developers.

🔊

CLI Sound Manager

Easily preview, list, map, add, and disable sounds using our custom script sounds.mjs. Change configurations in one command.

⚡

Zero NPM Dependencies

Pure Node.js implementation. Uses system-native media players: afplay on macOS, PowerShell on Windows, and common engines on Linux.

🔔

Desktop Alerts

Displays native desktop notifications concurrently with sound playbacks. Click notifications to focus back on your terminal.

🛠

Multi-IDE & Agent Support

Out of the box templates for Claude Code, Gemini CLI, Codex, and GitHub Copilot. Extendable to any tool that fires post-execution hooks.

🔄

Idempotent Installer

Re-running the installation script updates mappings without duplicating lines or messing up existing JSON configurations.

🏰

Failsafe Audio Playback

If the designated sound files are missing or the audio command fails, the notifier falls back gracefully without interrupting your agent's turns.

Ready for Action?

Run the installation script in your terminal to bring Warcraft III worker notifications to your local development environment.
